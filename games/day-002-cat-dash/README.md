# Cat Dash

**Day 002** - 1 Game Per Day project by Eric Canales.

Fast-paced pixel-art side-scrolling platformer / infinite runner starring a cute cat.

## Gameplay
- Left / Right (or A/D): move the cat
- Space: normal jump
- Ctrl: stop the cat + charge a powerful super jump (release for big vertical + forward boost)
- Reach the glowing flag goal platform to complete each level and advance
- Mix of wide horizontal platforms and tall vertical pillars
- Quick action with responsive physics and procedural generation

## Controls (in-game)
Reach the goal before falling off the bottom. Super jumps help cross big gaps or reach high platforms.

Pixel art cat and platforms are drawn directly (chunky retro sprite style).
